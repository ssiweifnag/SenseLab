# SensesLab 實驗室：基於 Home Assistant 的能源管理系統驗證場域

**核心方案**：

使用 **ESP8266 搭配 PZEM-004T 和內部溫濕度感測器**，在實驗室環境中驗證智慧能源管理系統的效能，並與傳統數位電表及數位鉤錶進行系統性比較。

---

### **1. 系統架構與應用場景**

### **(1) ESP8266 + PZEM-004T 方案**

- **系統設計**
    - **能源監測**：透過 PZEM-004T 測量電壓（V）、電流（I）及功率（P）。
    - **環境監測**：搭載溫濕度感測器（如 DHT11），監控設備內部環境參數。
    - **平台整合**：使用 ESP8266 上傳數據至 Home Assistant，進行能耗即時分析與展示。
- **應用場景**
    - 智慧家庭能源管理（能耗監控與節能預測）。
    - 教育實驗場域，研究能源管理與 IoT 系統整合。
    - 綠建築項目，用於環境感測與能源優化。

### **(2) 傳統數位電表**

- **系統設計**
    - 固定設備，主要用於準確計量電力使用量，數據本地存儲。
    - 支援計費功能，廣泛應用於住宅與商業設施。
- **應用場景**
    - 長期穩定的能源監測與計費。
    - 商業應用，例如工業設施或大型電力系統。

### **(3) 數位鉤錶**

- **系統設計**
    - 手持設備，通過鉤表感測器進行即時電流檢測，適用於快速測試。
- **應用場景**
    - 現場設備故障排查與短期診斷。
    - 適合檢測單一回路或臨時監測場景。

---

| --- | --- | --- | --- |
| **應用範圍** | - 家庭與中小型場域能源監控 | - 長期穩定計量，主要應用於商業與住宅能源管理 | - 臨時監測或現場檢修 |
| **未來性** | - 高，結合 IoT 與 AI，適合智慧建築與綠能應用 | - 中，穩定性強但功能固定，應用場景侷限 | - 低，專注即時測試，無長期監測能力 |
| **擴展性** | - 高，支持多感測器與數據整合 | - 無，固定功能設計 | - 無，功能簡單，無法擴展 |

---

### **3. 準確度比較**

| **測量指標** | **ESP8266 + PZEM-004T** | **傳統數位電表** | **數位鉤錶** |
| --- | --- | --- | --- |
| **電壓精度** | ±0.5%（受 PZEM 模組版本影響） | ±0.2% 至 ±0.5%（符合 IEC 62053 標準） | ±1%（偏向大電流測試） |
| **電流精度** | ±1%（CT 感測器誤差） | ±0.5% 至 ±1% | ±2%（適用於高電流） |
| **功率精度** | ±1%（需進行校準和補償） | 高，適用於長期穩定計量 | 低，僅供即時檢測使用 |
| **環境穩定性** | - 易受溫度影響，需溫度補償 | - 高穩定性，內建抗干擾設計 | - 容易受外部磁場與電磁干擾 |

**分析：**

- **傳統數位電表**的穩定性與準確度最佳，適合長期計量與商業用途。
- **ESP8266 + PZEM-004T**方案足以滿足智慧家庭與中小場域需求，但需進行校準與數據補償。
- **數位鉤錶**適合臨時測試，準確度與穩定性最低。

### **5. 實驗室驗證價值**

SensesLab 作為智慧建築的研究驗證場域，結合 Home Assistant 平台與 ESP8266 + PZEM-004T 的模組化架構，具有以下價值：

### **(1) 研究價值**

- **能源管理研究**：探討不同設備在多場景下的能源監測精度與穩定性。
- **智慧建築應用**：測試能耗數據與環境參數之間的關聯，為綠建築優化提供依據。
- **IoT 系統整合**：結合 MQTT 協議與數據視覺化工具，開發具有實驗與應用價值的能源管理系統。

### **(2) 教學價值**

- **實作學習**：學生可參與 IoT 設備部署、數據分析與應用開發。
- **跨領域結合**：結合建築與能源學科，培養智慧建築與節能設計的專業技能。

### **(3) 實驗室特色**

- 建立多設備對比測試平臺，驗證系統效能與實用性。
- 以低成本高效能方案為主，適合推廣至中小型智慧建築或家庭能源管理應用。

---

### **6. 總結**

- **ESP8266 + PZEM-004T 方案**適用於智慧能源管理，具備成本低、靈活性高與數據可視化等優勢，適合在 SensesLab 實驗室進行智慧建築應用測試。
- 與傳統數位電表相比，雖精度略低，但更具擴展性與未來應用潛力。
- 數位鉤錶則限於短期測試用途，無法滿足長期監控需求。
